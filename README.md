# Spliced-Images-Detection
Image Splicing Detection using CNN and Error Level Analysis

This project presents an image splicing detection system designed to identify cut-and-paste image forgeries using deep learning and forensic image analysis techniques. Image splicing involves copying a region from one image and inserting it into another image, resulting in manipulated content with subtle inconsistencies. The proposed system combines convolutional neural networks with Error Level Analysis (ELA) to enhance the detection of compression anomalies and tampered regions. A ResNet-based architecture with attention mechanisms is used to improve feature learning and focus on suspicious image regions. The model is trained on facial manipulation datasets including DeFacto-Face and custom splicing datasets containing authentic and manipulated facial images. Image preprocessing techniques such as resizing, normalization, and augmentation are applied to improve model performance and generalization. The system is implemented using Python, PyTorch, OpenCV, NumPy, Matplotlib, and Scikit-learn libraries. The performance of the model is evaluated using Accuracy, Precision, Recall, ROC-AUC, Confusion Matrix, and Grad-CAM visualization techniques. The project demonstrates effective detection of spliced and manipulated images and can be further extended for advanced digital image forensics and real-time forgery detection applications.
